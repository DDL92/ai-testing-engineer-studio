# Message Library

All templates are manual-review only. Do not auto-send. Personalize using only reviewed local lead data. Keep messages short, professional, and evidence-led.

## LinkedIn Connection

```text
Hi {{firstName}}, I work on QA automation and Playwright smoke test coverage for web products. I noticed {{company}} and thought it would be useful to connect.
```

## LinkedIn First Message

```text
Hi {{firstName}}, thanks for connecting. I help teams find release risks in signup, checkout, booking, dashboard, and API-heavy flows using focused QA audits and Playwright smoke tests.

If useful, I can take a quick manual look at {{company}} and suggest a small QA audit angle. No pressure.
```

## QA Audit Offer

```text
Hi {{firstName}}, I noticed {{company}} has {{observedWorkflow}}. That type of flow is often worth a focused QA review because small regressions can affect user trust.

I offer a lightweight QA Audit with screenshots, cautious findings, and recommended Playwright smoke test opportunities. Would it be useful if I sent a short outline?
```

## Follow-Up After No Response

```text
Hi {{firstName}}, quick follow-up. If QA automation is not a priority right now, no worries.

The most common starting point is a small audit of one public flow, then deciding whether Playwright coverage is worth adding.
```

## Discovery Call Invitation

```text
Hi {{firstName}}, if helpful, we can do a short discovery call to identify the top 1-2 workflows where QA coverage would reduce release risk.

I can keep it practical: current process, biggest regression risks, and whether an audit or Playwright Starter Pack makes sense.
```

## Manual Review Checklist

Before sending any message:

- Confirm the company and contact are correct.
- Remove unsupported claims.
- Remove pressure tactics.
- Confirm the offer matches the lead score.
- Confirm Daniel approves the final message.
