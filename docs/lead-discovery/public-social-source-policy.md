# Public Social Source Policy

This policy defines which public social and community sources may be used for AI Lead Discovery Studio query planning and safe public search.

## Allowed Sources

- Public Reddit threads that are indexed and accessible without login.
- Public Facebook posts or groups only when indexed and accessible without login.
- Public Instagram posts or pages only when indexed and accessible without login.
- Public TikTok posts or pages only when indexed and accessible without login.
- Public forums with visible indexed discussion pages.
- Public event request and event planning boards.
- Public RFP boards.
- Public community recommendation threads.

## Disallowed Sources

- Private groups.
- Login scraping.
- Profile harvesting.
- Mass messaging.
- Auto-DMs.
- Contact extraction.
- Hidden or private data.
- Aggressive crawling.
- Attempts to bypass platform restrictions.

## Operating Boundary

Social-source expansion is query planning and safe public search only. The system may store public result URLs, titles, snippets, query metadata, and source metadata. It must not log in, scrape private content, crawl profiles, extract contact details, automate outreach, submit forms, send DMs, or message leads.

Human review is required before any candidate is enriched, delivered, verified, or used in a client-facing workflow.
