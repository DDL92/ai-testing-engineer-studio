# Automation Strategy

## Automate With Codex

- Generate initial POM classes from application flow notes.
- Draft test scenarios from bug reports and acceptance criteria.
- Create client reports from structured findings.
- Refactor brittle selectors into stable Playwright locators.
- Produce handoff READMEs and CI recommendations.

## Automate With Playwright

- Login smoke tests.
- Critical user flows.
- API health checks.
- AI response contract checks.
- Regression tests for fixed bugs.
- Screenshots, traces, and videos on failure.

## Automate With GitHub Actions

- Pull request smoke tests.
- Nightly regression smoke suite.
- HTML report artifact upload.
- Trace/video retention on failure.
- TypeScript typecheck.

## Automate With Templates

- Intake forms.
- Audit reports.
- Coverage matrices.
- Final delivery messages.
- Retainer proposals.
- Upwork proposals.

## Do Not Automate Yet

- Lead scraping.
- CRM workflows.
- Invoice automation.
- AI scoring platform.
- Large dashboard.
- Multi-tenant product flows.

## Fast Delivery Flow

1. Intake.
2. Identify top three business-critical flows.
3. Build or adapt POM.
4. Add 1-3 smoke tests and one API health check.
5. Run locally and capture artifacts.
6. Deliver report, matrix, and next-step offer.
