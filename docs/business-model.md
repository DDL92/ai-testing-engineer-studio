# Business Model

## Revenue Model

The MVP earns revenue through fixed-scope services first, then monthly retainers.

## Services

| Service | Price | Role |
| --- | ---: | --- |
| Playwright QA Automation Audit | $300-$500 | Entry offer |
| Playwright Starter Framework | $800-$1,500 | Main cash-flow offer |
| AI Testing Audit | $500-$1,000 | Differentiated offer |
| Monthly QA Automation Retainer | $750-$2,500 | Recurring revenue |

## Future Products

- Playwright audit checklist.
- AI testing checklist bundle.
- Starter framework template.
- GitHub Actions QA template.
- Prompt injection defensive test pack.

## Simple Funnel

Content or Upwork job -> fixed audit -> framework build -> retainer -> product referral.

## Channels

- Upwork Project Catalog.
- Upwork proposals.
- LinkedIn posts.
- X posts.
- Instagram carousels.
- GitHub portfolio.
- Direct outreach to SaaS founders and agencies.

## Metrics

- Proposals sent per week.
- Reply rate.
- Audit calls booked.
- Paid audits closed.
- Framework conversion rate.
- Retainer conversion rate.
- Monthly revenue.

## Path To $3k-$5k Monthly

- Option A: two $1,500 frameworks + one $750 retainer.
- Option B: one $1,000 AI audit + two $1,500 frameworks.
- Option C: two $1,500 retainers + two $500 audits.

## Monetization Roadmap

1. Week 1: launch repo, assets, and offers.
2. Weeks 2-4: sell audits and collect proof.
3. Month 2: productize starter framework and retainer.
4. Month 3: add low-ticket templates from repeated delivery assets.
