# Lead Discovery Engine Roadmap

## v1 Complete

- Local seed catalog.
- Niche-based deterministic discovery.
- QA Automation scoring.
- Candidate storage under `data/leads`.
- Markdown reports under `output/leads`.
- Company pack generation by company name.
- Human approval boundaries.

- Add a manual candidate import format for Daniel-reviewed companies.
- Add duplicate review against `data/leads.json`.
- Add per-niche scoring weights.
- Add safer promotion checklist before `lead:add`.

## v1.2

- Add source quality reporting for discovered candidates.
- Add first-audit readiness ranking.
- Add follow-up prompts for manually approved leads only.

- CRM integrations.
- LinkedIn automation.
- Email automation.
- Scraping behind logins.
- Autonomous outreach.
