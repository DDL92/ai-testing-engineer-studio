# Weekly Executive Dashboard

Run:

```bash
npm run business:weekly
npm run business:weekly -- --days 14
npm run business:weekly -- --days 30
```

The dashboard writes:

- `sales-marketing-engine/operator/generated/weekly-executive-dashboard.md`
- `data/leads/weekly-executive-dashboard.json`

## How To Read It

Start with the executive summary. It identifies the current business status, the main bottleneck, the biggest opportunity, and the recommended focus for next week.

The most important numbers are projected MRR, progress to $3,000/month, top revenue opportunities, leads needing enrichment, audit-ready leads, and next 10 highest-value actions.

## Conversion Rates

Early conversion rates can be misleading. If the sample size is small, use the dashboard as a directional signal, not a conclusion. Focus on building consistent volume and clean status tracking.

## Next 10 Highest-Value Actions

The action list prioritizes higher estimated revenue and lead score. Daniel should review the command, open the lead review, and only then enrich, audit, send, close, or convert manually.
