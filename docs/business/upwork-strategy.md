# Upwork Strategy

Do not depend on Upwork. Use it selectively for high-score opportunities where the scope and budget justify paid connects.

## Good Jobs

- Playwright
- Cypress
- Selenium
- TypeScript
- CI/CD
- Long-term QA Automation
- SDET
- API testing

## Avoid

- Very low budget
- Manual QA only
- No payment verification
- Unclear scope
- Too many proposals
- Cheap one-off testing

## Proposal Positioning

Start with a focused audit or starter framework. Keep the promise practical: critical flows, stable Playwright tests, CI reporting, and clear handoff notes.
