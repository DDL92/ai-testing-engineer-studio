# Weekly Review SOP

Run the weekly dashboard every Monday:

```bash
npm run business:weekly
```

## Monday Routine

1. Review `weekly-executive-dashboard.md`.
2. Review `revenue-summary.md`.
3. Review `pipeline-summary.md`.
4. Review `source-quality-report.md`.
5. Check lost reasons.
6. Review approval queue drafts.
7. Pick the next 10 highest-value actions.
8. Enrich top leads manually.
9. Run audits for qualified leads.
10. Send approved messages manually.
11. Close bad leads.

## What Matters Most

- MRR progress toward $3,000-$5,000/month.
- Number of high-fit leads ready for audit.
- Number of leads blocked by missing contact data.
- Conversion-ready leads.
- Lost reasons that can be fixed by better targeting or messaging.
- Source quality signals that show which sources should be kept, improved, or disabled.

## Goal Support

The weekly dashboard keeps Daniel focused on revenue-producing work: finding qualified leads, converting audit interest into paid offers, and growing monthly QA maintenance revenue.
