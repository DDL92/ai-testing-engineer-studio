# 7-Day Launch Plan

## Day 1: Repo + Offers

Objective: Make the MVP understandable and sellable.  
Tasks: Review README, service offers, and pricing.  
Files: `README.md`, `docs/service-offers.md`, `offers/*`.  
Codex: Tighten copy and remove scope creep.  
You: Pick one primary offer and publish it.  
Result: Clear first offer.  
Success: One public offer ready to send.

## Day 2: Playwright Framework

Objective: Make the framework runnable and client-adaptable.  
Tasks: Install deps, run typecheck, test against demo/client app.  
Files: `playwright-framework/*`, `playwright.config.ts`.  
Codex: Adapt POM to target app.  
You: Provide app URL and test credentials.  
Result: First smoke suite.  
Success: At least one test runs.

## Day 3: AI Testing Mini-Kit

Objective: Package the AI audit differentiator.  
Tasks: Review checklists, customize examples, create sample scoring.  
Files: `ai-testing-kit/*`.  
Codex: Generate client-specific AI test matrix.  
You: Pick an AI niche to target.  
Result: AI audit deliverable.  
Success: One AI audit sample ready.

## Day 4: Client Delivery System

Objective: Be ready to deliver a paid audit.  
Tasks: Prepare intake, report, matrix, delivery message.  
Files: `client-delivery-system/*`.  
Codex: Convert findings into report language.  
You: Save templates for repeated use.  
Result: Delivery workflow.  
Success: Client can receive a professional package.

## Day 5: Upwork + Proposals

Objective: Start selling.  
Tasks: Update profile, publish catalogs, send proposals.  
Files: `upwork-assets/*`.  
Codex: Customize proposals to live job posts.  
You: Send five proposals.  
Result: Outreach live.  
Success: Five targeted proposals sent.

## Day 6: Content + GitHub Portfolio

Objective: Build credibility.  
Tasks: Publish first post and carousel, push repo.  
Files: `content-engine/*`, `README.md`.  
Codex: Repurpose technical assets into posts.  
You: Post and pin repo.  
Result: Public proof.  
Success: One post, one carousel, one pinned repo.

## Day 7: Outreach + Launch

Objective: Create first sales conversations.  
Tasks: Send direct messages, follow up, track replies.  
Files: `docs/commercial-mvp-audit.md`, `upwork-assets/proposal-templates.md`.  
Codex: Personalize outreach.  
You: Contact 20 prospects.  
Result: First pipeline.  
Success: 20 prospects contacted and replies tracked.
