# Do Not Run Until Credits Reset

The following commands may consume Tavily credits or exercise provider paths. Do not run them while the system is paused for `cost_budget_paused`.

- `npm run leads:search`
- `npm run leads:morning`
- `npm run leads:daily`
- `npm run leads:test-provider`

Use `npm run leads:safe-commands` to print the current safe offline command list.
