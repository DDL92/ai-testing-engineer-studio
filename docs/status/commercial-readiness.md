# Commercial Readiness

Flora and Fauna Foods is commercially prepared for a manual conversation and paid pilot discussion. Current delivery remains preparation-only until approved leads exist.

## Offer Pack Status

- Status: ready
- Command: `npm run leads:offer-pack`
- Output: `output/client-offer/`
- Includes service explanation, lead definitions, exclusions, pricing, deliverables, and terms.

## Meeting Prep Status

- Status: ready
- Command: `npm run leads:meeting-prep`
- Output: `output/commercial/`
- Includes pitch, agenda, pilot explanation, objections, closing questions, follow-up drafts, and value story.

## Pilot Pack Status

- Status: ready as a generator
- Command: `npm run leads:pilot-pack`
- Output: `output/pilot-pack/`
- Current expected state while paused: `NO_DELIVERY` when no approved reviewed leads exist.

## Delivery Router Status

- Command: `npm run leads:delivery-router`

- Supports Pilot Package, Growth Package, and Premium Package delivery plans.
