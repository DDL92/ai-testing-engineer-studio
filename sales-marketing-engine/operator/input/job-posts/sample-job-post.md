# Sample Upwork Job Post

Title: Playwright QA Automation Engineer Needed for SaaS Regression Suite

We are a B2B SaaS company looking for a QA automation engineer to help us set up reliable Playwright tests for our staging app. Our team currently does manual regression before each release, and it is slowing us down.

Scope:

- Review our login, dashboard, user management, and billing flows.
- Set up Playwright with TypeScript.
- Use Page Object Model.
- Add smoke tests that can run in CI.
- Include HTML reporting and screenshots/traces on failure.
- Help us understand what to automate first.

Nice to have:

- API health checks.
- GitHub Actions experience.
- Ability to explain the framework to developers.

Budget: $800-$1,500  
Timeline: 1 week  
Access: staging environment and test user available
