# Content Quality Checklist

Before publishing:

- [ ] Clear hook.
- [ ] Specific audience.
- [ ] One core idea.
- [ ] Practical takeaway.
- [ ] No generic AI hype.
- [ ] CTA included.
- [ ] Tied to one service.
- [ ] Reusable across platforms.
- [ ] Simple language.
- [ ] Technical credibility.
- [ ] No exaggerated claims.
- [ ] No client-sensitive details.
