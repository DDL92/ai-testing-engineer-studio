# Weekly Sales Dashboard

Use this folder to track sales performance every week and decide what to improve.

## What Numbers Matter

- Leads added.
- Messages sent.
- Replies.
- Calls booked.
- Proposals sent.
- Deals won.
- Revenue closed.
- Follow-ups completed.

## Friday Review

Every Friday:

1. Update `sales-metrics-tracker.csv`.
2. Fill `weekly-sales-dashboard.md`.
3. Complete `weekly-review-template.md`.
4. Compare activity against the monthly revenue plan.
5. Choose one sales experiment for next week.

## How To Decide What To Improve

- Low replies: improve targeting or first message.
- Replies but no calls: improve CTA or qualification.
- Calls but no proposals: improve discovery questions.
- Proposals but no wins: improve pricing, proof, or scope clarity.
