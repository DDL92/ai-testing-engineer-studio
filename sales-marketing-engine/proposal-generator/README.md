# Proposal Generator

This folder helps create fast, customized proposals for Upwork, LinkedIn, cold email replies, referrals, and direct leads.

## How To Use

1. Read the lead context or job post.
2. Choose the best offer:
   - Unclear QA problem: Playwright QA Automation Audit.
   - Known need for automation setup: Playwright Starter Framework.
   - AI feature quality/safety risk: AI Testing Audit.
   - Ongoing test work: Monthly QA Automation Retainer.
3. Use the master prompt or service-specific template.
4. Customize the first line with the client's actual context.
5. Keep the proposal short, specific, and outcome-focused.

## How To Avoid Generic Proposals

- Mention one specific detail from the job/company.
- Name the likely pain point.
- Recommend one clear first step.
- Include 1-2 smart questions.
- Avoid long bios and vague claims.
- Do not overpromise timelines or outcomes.
