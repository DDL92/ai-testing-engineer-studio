# Client Delivery Prompts

## Create README Client

Uso: Client handoff.  
Prompt: Create a concise README for a Playwright client project. Include install, env setup, test commands, report commands, folder structure, how to add tests, and troubleshooting.  
Resultado esperado: Client-ready README.

## Create Final Report

Uso: Audit delivery.  
Prompt: Convert these QA findings into a professional automation audit report with executive summary, risks, recommended tests, quick wins, 7/30/60-day roadmap, and next-step proposal.  
Resultado esperado: Delivery report.

## Convert Bugs Into Tests

Uso: Regression coverage.  
Prompt: Convert these bug reports into Playwright regression test scenarios. Include preconditions, steps, assertions, priority, and recommended automation type.  
Resultado esperado: Test scenario list.

## Propose Retainer

Uso: Upsell.  
Prompt: Create a concise monthly QA automation retainer proposal with three tiers: $750, $1,500, and $2,500. Tie each tier to the client's current risks and automation needs.  
Resultado esperado: Retainer proposal.
