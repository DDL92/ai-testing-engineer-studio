# Studio Rebuild Guide

Generated: 2026-06-13T20:49:55.325Z

1. Fresh Mac: install macOS updates and create a local development folder.
2. Install Node: use the project-compatible Node version available on the machine.
3. Install Dependencies: run npm install from the repo root.
4. Clone Repo: clone the GitHub repository or restore the full repo folder from backup.
5. Run Validation: run npm run typecheck and npm test.
6. Generate Dashboard: run npm run dashboard:generate.
7. Verify Mobile: run npm run dashboard:mobile and open the same-WiFi URL on a phone.
8. Verify Outputs: run npm run studio:inventory and npm run studio:output-audit.
9. Verify Studio Health: run npm run studio:health, npm run studio:release-check, and npm run studio:recovery-check.
